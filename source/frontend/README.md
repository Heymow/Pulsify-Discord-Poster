# Discord Poster Frontend

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Backend Setup

Ensure the backend is running on port 3000.
```bash
cd ../backend
node server.js
```

## Features

- **Dashboard**: Send posts to Discord channels.
- **Channels**: Manage your channel lists.
- **Settings**: Connect your Discord account.
